Shortcuts:

- P : selecciona una figura
- L : se prepara para dibujar una linea
- Q : se prepara para dibujar un rectangulo
- C : se prepara para dibujar un circulo
- E : se prepara para dibujar una elipse
- T : se prepara para dibujar un triangulo
- F : (Toggle) si es una figura que se le puede colocar relleno le coloca relleno, se puede prender y apagar pero debe seleccionar la figura previamente con P. Si esta en modo curva de bezier (B) deja de tomar puntos de control con el click y crea una nueva
- B : se prepara para dibujar una curva de bezier
- M : (Toggle) si hay una figura seleccionada, coloca el modo "Mover" para desplazar la figura a donde se de click. Se debe presionar nuevamente para quitar el modo Mover
- 1 : trae al frente una figura
- 2 : trae adelante una figura
- 0 : trae al fondo una figura
- 9 : trae atras una figura
- D : elimina la figura seleccionada



falta: subdividir curva de bezier y guardar y cargar figuras
